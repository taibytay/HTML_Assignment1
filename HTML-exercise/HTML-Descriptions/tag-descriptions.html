Common HTML Tags

<iframe> </iframe>

  • Purpose: shows another web page within a page
  • Example: <iframe src="https://en.wikipedia.org/wiki/Family_business" width="200" height="250"> </iframe>
  • Attributes:
    • Attribute Value Meaning
    src URL sets the page to be displayed in the iframe
    height number sets the height of the iframe
    width number sets the width of the iframe
    frameborder number set the border around the iframe
<link>

  • Purpose: links an HTML document to a CSS stylesheet
  • Example: <link rel="stylesheet" href="/css/profile1.css">
  • Attributes:
    • Attribute Value Meaning
    rel stylesheet sets the type of link
    href URL sets the address of the file to link to
<input>

  • Purpose: adds an input field where a user enters data
  • Example: <input type="text" maxlength="5">
  • Attributes:
    • Attribute Value Meaning
    name text_or_numerical_input sets the name and tells the form to send this input to the server when the form is submitted
    maxlength number sets maximum number of characters to be entered
    value text_or_numerical_input sets the text showing in or on the input
    type name_of_type determines the type of input
    radio adds a radio button where a user can make one selection from multiple choices
    checkbox adds a check box in a list where a user can select any number of check boxes for choices
    range adds a range slider element
    password adds an element where a user can enter a password with the text hidden as it is typed
    button adds a button element
